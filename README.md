# NotesApp
This is a full project to saved personal notes. It has many features: 
 - Login
 - Signup
 - Create Notes
 - Edit Notes
 - Delete Notes
 
Running on Node.js
- Express (Endpoints) 
- Express-handlebars (Frontend)
- MongoDB (Database)
- Mongoose (Data modeling)
- Passport (Authentication)
- Bcryptjs (Password hashing)
## Installing
Install dependencies.
```
npm install 
```
## Script
Run the project.
```
npm run dev
```
